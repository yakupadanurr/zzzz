- **Ana sayfa**

## Yayına Alma
1. [Vercel](https://vercel.com) hesabı açın.
2. Bu projeyi GitHub'a yükleyin.
3. Vercel üzerinden "Import Project" diyerek bu repo'yu seçin.
4. Deploy işlemi tamamlandığında siteniz yayında olacaktır.
